  3. Combinator มีกี่แบบอะไรบ้าง

    4. Combinator
    Combinator exposition
    div p เข้าถึง Tag p ทุก Tag ที่อยู่ใน Tag div
    div > p เข้าถึง Tag p ทุก Tag ที่อยู่ใน Tag div
    div + p เข้าถึง Tag p ตัวแรกที่อยุ่ ถัดจาก Tag div
    div ~ p เข้าถึง Tag p ทุกตัวที่อยุ่ ถัดจาก Tag div

  4. Selecter มีกี่แบบอะไรบ้าง

    5. Selecter
    Selecter exposition
    .class เข้าถึง Tag ที่มี class="...."
    .class1 .class2 เข้าถึง Tag ที่มี class="...." ทั้ง class1 และ class2
    #id เข้าถึง Tag ที่มี id="...."
    * เข้าถึง ทุก Tag
    element เข้าถึง tag elemenet ที่กำหนด
    element.class เข้าถึง tag elemenet ที่กำหนด และ tag ที่มี class="...."
    element1,element2 เข้าถึง tag elemenet1 และ element2
    element1,element2 เข้าถึง tag elemenet1 และ element2
    [attribute] เข้าถึง Tag ที่มี attribute
    [attribute = value] เข้าถึง Tag ที่มี attribute ที่มีค่าว่า ...
    [attribute ~= value] เข้าถึง Tag ที่มี attribute ที่มีค่าว่า ...
    [attribute |= value] เข้าถึง Tag ที่มี attribute ที่ข้างหน้ามีค่าว่า ...
    [attribute ^= value] เข้าถึง Tag ที่มี attribute ที่ข้างหน้ามีค่าว่า ...
    [attribute $= value] เข้าถึง Tag ที่มี attribute ที่มีค่าว่า ...
    [attribute *= value] เข้าถึง Tag ทุกตัวที่มี attribute ที่มีค่าว่า ...
