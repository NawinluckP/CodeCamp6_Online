• Value ใน display property มีอะไรบ้าง

    • DisplayProperty
    Display ความหมาย
    none ไม่แสดงเอเลเมนที่กำลังใช้
    bock ขึ้นบรรทัดใหม่เมื่อมีการเปิด Tag นี้
    inline ไม่ขึ้นบรรทัดใหม่เมื่อมีการเปิด Tag ที่มี display นี้
    inline-bock ข้างใน Tag แสดงผลแบบ bock ข้างนอก แสดงแบบ inline

  • Grid และ flex แตกต่างกันอย่างไร

    • Grid-Flex
    display ความต่าง
    flex ใช้ในการจัด layout ทั้งคุ่แต่ Flex ไม่สามารถ ยุบรวม tag ได้เหมือน grid
    grid ใช้ในการจัด layout ทั้งคุ่แต่ Flex ไม่สามารถ ยุบรวม tag ได้เหมือน grid

  • Justify and align แตกต่างกันอย่างไร

    • Grid-Flex
    display ความต่าง
    Justify จัดแถวแนวตั้ง
    align จัดแถวแนวนอน
